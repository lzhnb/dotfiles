# dotfiles

## Features
- [XDG base directories](https://specifications.freedesktop.org/basedir-spec/basedir-spec-latest.html) support

## Applications
- zsh
- tmux
- fzf
- vim
- neovim
- ranger
- miniconda3 (please setup the miniconda3 in `$XDG_CONFIG_HOME(~/.local/share)`)
